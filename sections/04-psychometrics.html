Construct

Start with a construct

Developing an assessment

  • Design an exam or survey to assess your construct
    • Each question is called an item in psych-talk
  • Is your assessment good?
    • Reliability: is the measurement consistent?
    • Validity: does it measure what it’s supposed to?

Reliability

If I measure individuals with the same trait, do they score the same?

  • Test-retest reliability: if you measure someone twice… do they get the same score
  • Internal consistency: does each question positively correlate with total score?
    • Chronbach’s Alpha: \(\alpha = \frac{k}{k - 1} \left( \frac{\sum_{i\neq j} \sigma^2_{y_iy_j}}{\sigma^2_x} \right)\)
  • Inter-Rater Reliability: if two people are scoring an assessment

Validity

Many types of validity (Coopers lists 12):

  • Content Validity: are the items a representative sample of the domain they measure
  • Criterion-Related Validity: how well do test scores correlate with an external, “real-world” outcome (the criterion)
  • Construct Validity: the extent to which a test measures the theoretical construct it was designed to measure

Factor Analysis & PCA

  • PCA: take correlation matrix \(\to\) compute SVD
  • Factor Analysis:
    • Assume some generative model: \(\mathbf{x} = \mathbf{\Lambda} \mathbf{f} + \boldsymbol{\epsilon}\)
      • \(\mathbf{x}:\) data
      • \(\mathbf{\Lambda}:\) loadings (constant across all individuals)
      • \(\mathbf{f}:\) factor scores
      • \(\boldsymbol{\epsilon}:\) error
    • Partition variance: \(\mathbf{\Sigma} = \mathbf{\Lambda} \mathbf{\Lambda}^T + \mathbf{\Psi}\)
    • Note \(\mathbf{\Lambda}\) not unique so we must “rotate”

The Latent Variable Assumption

What polychoric/tetrachoric actually assume

  • Observed categories = discretization of a hidden continuous variable
  • Latent variable assumed to be normally distributed

The Latent Variable Assumption

Example: “Rate your pain 1–5”

  • Your actual pain exists on a continuous scale
  • You cross a threshold → you report the next category up

Hidden number line:

——-|——-|——-|——-|——-

1 2 3 4 5

Thresholds (\(\tau\)) carve it into bins

  • We only observe which bin
  • Latent value is inferred
  • Polychoric recovers the latent correlation

Item Response Theory (IRT)

Models probability of a correct/positive response

Two key parameters per item:

  • Difficulty (β): where on ability scale does P(correct) = 0.5?
  • Discrimination (α): how steeply does the curve rise?

Item Characteristic Curve (ICC):

\[P(\text{correct} \mid \theta) = \frac{1}{1 + e^{-\alpha(\theta - \beta)}}\]

S-shaped logistic curve, θ = person’s latent ability

IRT: Why It Matters

Advantages over classical test theory

  • Item parameters don’t depend on the sample
  • Person ability doesn’t depend on which items used
  • Used by: GRE, SAT, LSAT, AP exams

A student taking a harder test version gets a fair ability estimate

The Big Caveat

All of this assumes (at least latent) continuity

  • What if variables are truly categorical?
    • Not discretized — genuinely unordered
    • Polychoric/IRT assumptions break down
  • Correlations and factor loadings become misleading
    • Artificially inflated or deflated
    • Factors may reflect response style, not constructs

Open Questions

  • Is “strongly agree → agree” the same distance as “agree → neutral”?
  • When does misspecification affect conclusions enough to matter?
